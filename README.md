# create-kirari

CLI tool to create a new Kirari blog project.

## Usage

```bash
# npm
npm create kirari@latest

# pnpm (recommended)
pnpm create kirari

# yarn
yarn create kirari

# npx
npx create-kirari
```

## What it does

1. Prompts for project name
2. Selects package manager
3. Clones the Kirari theme template
4. Installs dependencies (optional)
5. Initializes git repository (optional)
